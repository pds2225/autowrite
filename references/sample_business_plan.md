# 2026년 재도전성공패키지 (예비)재창업기업 사업계획서

## 과제 개요

과제명: AI·데이터 기반 수출지원 One-Stop 플랫폼 (수출국가추천+바이어매칭+수출계약)
기업명: 예비재창업자

### 아이템(서비스) 개요
1. 폐업 원인: 기존 컨설팅 사업의 구조적 한계로 인한 매출 부진
→사업구조(노동집약) 시장특성(경험기반) 수익구조(일회성)
2. 해결 방안: AI·데이터 기반 수출지원 One-Stop 플랫폼
→SaaS기반 AI/데이터활용 수익구조(지속성)
3. 사업필요성: 중소기업 수출의 어려움
- 대규모 수출 데이터 파편화, 담당자 없이 정보 수집 실질적 불가능
- 중소기업의 낮은 AI 도입률, 데이터 활용 방안 부재
- 정부지원사업은 바이어 정보 제공에 그쳐 실계약 전환율 저조
4. 아이템: 단계별 수출지원(국가선정>바이어발굴>수익분석>계약체결)
1단계: 수출국가 추천
2단계: 저품질·저적합 바이어 필터링
3단계: 수출 수익성 검증 시뮬레이션
4단계: 자동계약서 생성 및 체결
MVP 개발완료 (Github 계정 및 메인코드)

## 1. 문제인식(Problem)

### 1-1. 폐업 원인 분석 및 개선 방안

가. 폐업 원인 분석 : 중소기업 컨설팅 서비스
컨설팅 서비스 폐업 원인별 개선 방안

(사업 구조) 노동집약적 서비스→SaaS 기반 자동화 플랫폼
- (노동력 기반 사업) 기존 컨설팅은 인력 충원 없이는 사업 확장 제한적
- (SaaS 기반 자동화) 인력 충원 없이도 원활한 사업 확장 가능

(시장 특성) 저연차 컨설턴트의 경험 부족→AI·데이터 활용으로 보완
- (경쟁력 확보한계) 고연차 컨설턴트의 경험량을 단시간에 따라잡기 힘듦
- (AI·데이터 활용) 컨설팅 경험을 데이터화, AI학습으로 경험량 격차극복

(수익구조) 일회성 수익모델→구독형 수익모델로 반복 매출
- (계약 매출) 목적 달성 후 종료되는 단발성 계약 형태로 재계약 어려움
- (구독료 매출) 구독모델도입으로 반복 구매 및 재계약 쉬움

나. 개선 방안: AI·데이터 기반 중소기업 수출지원 One-Stop 플랫폼
재창업 아이템: 중소기업 수출지원 플랫폼
- 수출국가 추천→바이어매칭→계약까지 수출 계획·실행을 한 곳에서 가능
- SaaS 기반으로 사업 확장성과 구독모델로 반복적 매출구조동시 확보

### 1-2. 재창업 아이템 목표시장(고객) 현황 및 필요성

가. 사업 필요성: 중소기업 수출의 높은 진입장벽
중소기업의 수출 단계별 진입 장벽과 해결방안 필요

1) 대규모 수출 데이터 파편화 →수출 데이터 통합시스템 필요
- 대량의 수출 데이터가 UNComtrade, KITA 등에 광범위하게 퍼져있어, 담당 인력이 부족한 중소기업은 데이터 수집조차 힘듦

2) 데이터 해석, 활용방안 부재 →AI·데이터 기반 서비스 필요
- 다양한 수출 데이터가 있어도 실제 수출 의사결정에 활용하기 어려움
- 데이터 해석과 활용을 자동화하는 AI 기반 수출 전략 프로세스 필요

3) 정부지원에도 문제 지속 →실제 수출로 연결되는 One-Stop 솔루션 필요
- 바이어 발굴 서비스는 6개 기관에서 이미 제공 중
- 그러나 수출 애로 1위는 여전히 '적격 바이어 발굴'

나. 사업 아이템 소개: 실제 수출로 연결되는 One-Stop 수출 솔루션
- 수출국가선정>바이어발굴>수익분석>계약체결까지 하나로 통합

1단계: 수출국가 추천 및 제재국 필터링
- 품목 입력(자연어) →HS코드 매칭 →제한국가·품목 필터링 →결과 표시
- KOTRA API 기반 머신러닝 AI →실제 수출 가능 국가만 제공

2단계. 저품질·저적합 바이어 필터링
- MOQ (바이어 MOQ≥셀러 MOQ), 인증 검증 (셀러 보유⊇바이어 요구)
→실제 거래 조건이 맞는, 적합성 높은 바이어만 선별, 맞춤형 제공

3단계. 수출 수익성 검증 시뮬레이션
- 관세, 물류비 등을 반영한 Landed Cost 기반 수익성 계산(BEP 등)
→수출 실행 전 시뮬레이션을 통해 적자 수출 구조적으로 차단

4단계. 자동계약서 생성(RPA) 및 체결
- 사용자 입력 값으로 수출계약서 자동 생성(RPA): Alibaba.com 벤치마킹

다. 목표시장 현황
- SOM: 3개년 이내 매출 38억 원 목표(SAM 대비 점유율 2.3%)
- TAM (전체 시장): 5,370 억원 - 한국 전체 중소기업 × (구독 + 협력 수익) *중소기업 9.8만개 / 월 구독료 29만원 / 협력수익 200만원 기준
- SAM (유효 시장): 1,644 억원 - 예상 수요 기업 × (구독 + 협력 수익) *배상 수요기업(연매출 10억이하 중소기업) 30,000개 / 이하 동일 기준
- SOM (수익 목표): 38 억원 - 구독고객 × (구독 + 협력 수익) Y3 구독 고객 700개사 / 이하 동일 기준 SAM 점유율 약 2.3%

라. 차별성 및 경쟁사 분석
- 수출지원 서비스(공공): 단순 정보 제공에 그쳐 실무에 활용하기 어려움
- 수출지원 서비스(민간): 수출 과정 중 일부 기능만 보유, 여전히 파편화
- 당사 서비스: 국가선정부터 계약까지 한 곳에서 가능한 수출실무 통합플랫폼
- 추천 결과 Hard Gate 필터링, 수출 불가 국가 제거해 실패 확률 낮춤

## 3. 성장전략(Scale-up)

### 3-1. 재창업 아이템 비즈니스 모델
