# BizPlan Injector — 새 양식 적용 가이드

## 새로운 양식 파일을 받았을 때

### Step 1. 양식 파일을 templates/ 폴더에 복사
```
templates/새양식.docx
```

### Step 2. 구조 분석
```bash
python inject.py --analyze templates/새양식.docx
```
출력 예시:
```
📊 표 목록 (14개):
  표00: 2행  [목차 | 항목]
  표01: 6행  [일반현황 | 기업명 | ...]
  표02: 7행  [창업아이템명 | ...]
  ...
  표13: 3행  [비목 | 참고]

📝 주요 단락 (헤딩):
  [018] 1 -1 .창 업 아 이 템 배 경 및 필 요성
  [031] 1 -2.창 업 아 이 템 목표시 장...
```

### Step 3. 스켈레톤 생성
```bash
python inject.py --skeleton templates/새양식.docx --output content_new.json
```

### Step 4. content_new.json 편집
- `delete_tables`: 목차 표, 비목 참고 표 인덱스 입력 (보통 [0, 마지막번호])
- `table_cells`: 분석 결과의 표 인덱스/행/열 번호 참조하여 값 입력
- `sections`: 헤딩 키워드 (예: "1 -1", "2-1") 확인 후 내용 입력

### Step 5. 실행
```bash
python inject.py --template templates/새양식.docx --content content_new.json
```

## 주의사항
- 양식마다 표 인덱스가 다를 수 있습니다. 반드시 `--analyze` 먼저 실행
- 헤딩 키워드는 실제 XML에 저장된 텍스트 기준입니다 (띄어쓰기 포함)
- 파란색 안내문구는 자동 제거됩니다
